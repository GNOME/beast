

BseParam
--------
a parameter that can be get/set on an object.
each parameter has a parameter specification associated.

BseParamSpec*
-------------
a parameter specification similar to an object's class,
though for a given parameter type multiple parameter specs may exist.

BseObject
---------
implements basic structure operations, introduces class concept

BseObjectClass
--------------
a class associated with each objects instance.
a class is referenced by its object instances.
introduces per-object parameters.

BsePlugin
---------
a plugin handle that holds information about existing plugins.

A Plugin
--------
provides object and class implementations derived from BseEffect.
provides enum and flags definitions.
provides sample/song file loading and saving routines.

BseType
-------
type ids are provided for
objects & classes,
parameters,
method interfaces,
